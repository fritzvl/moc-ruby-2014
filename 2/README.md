Ruby programming language
---------------

* Ruby interpreters
  * MRI
  * Rubinius
  * JRuby
  * IronRuby
  * Maglev
  * Opal

* Key features
  * Object-oriented
  * Dynamic and duck typing
  * Everything is an expression
  * Creation of DSLs
  * Reflection and alteration
  * Lexical closures, iterators and generators, with a unique block syntax
  * Exception handling
  * GC
